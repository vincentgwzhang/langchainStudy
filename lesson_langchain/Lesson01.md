# LangChain 是什么？

LangChain 是一个构建基于大语言模型（Large Language Model，LLM）应用的框架。它提供了一组工具和抽象层，来简化和增强与语言模型进行交互和开发复杂应用的过程。

LangChain 的设计目标是帮助开发者高效地构建多功能、可扩展的自然语言处理（NLP）应用。
